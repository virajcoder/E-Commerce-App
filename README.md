# E-Commerce Website

A Ecommerce Website made with React.js Framework.


## Demo

https://vbeyfashion.vercel.app/

## Features

- Easy to integrate with Backend
- Fully Responsive

## Run Locally
Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* React
* Redux
* Bootstrap
* Fake Store API
